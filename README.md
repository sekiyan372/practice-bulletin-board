# yuwaku-admin

湯涌のフォトコンテストの管理者用ページ

# Version
- node: v16.13.1

## Getting Started

clone した時は 以下を行う。

```bash
# パッケージインストール
$ yarn install
# 開発環境のサーバー起動
$ yarn dev
```

サーバー起動後 [http://localhost:3000](http://localhost:3000) にブラウザでアクセスする。
